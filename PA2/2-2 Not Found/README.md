# CST2021F 2-2 Not Found

------

### **描述**

　　给你输入一个 01 串 A。你需要输出一个最短的 01 串 B，使得 B 在 A 中从未出现过。如果有多个可行的解，你需要输出字典序最小的那一个。

### **输入**

　　共计一行，包含这个 01 串 A。注意输入文件将以换行符结尾。

### **输出**

　　共计一行，包含你找到的 01 串 B。建议输出文件以换行符结尾。

### **输入样例**

```
10100011
```

　　*此样例是第 1 个测试点。

### **输出样例**

```
110
```

### **限制**

　　1 ≤ A 的长度 ≤ 16,777,216

　　时间：1 sec

　　空间：6 MB
